# 更新说明

## v1.2.0 (2026-05-26)

- **LLM 文本润色** — 集成 Vercel AI SDK，支持 DeepSeek、OpenAI、Anthropic、Gemini、OpenRouter、硅基流动、Ollama、自定义 OpenAI 兼容共 8 家厂商，可对 ASR 识别结果进行 LLM 后处理（格式化、润色、翻译等）。
- **润色模板管理** — 新增 `prompts.json` 管理多套提示词模板，每个模板可独立绑定快捷键与触发模式，实现不同场景下的润色需求。
- **实时音频波形** — 录音期间在浮动窗口中显示实时音频波形动画，提供直观的录音反馈。
- **提示音** — 录音就绪与识别成功时分别播放开始音和结束音，直观感知录音状态。
- **DMG 构建格式** — macOS 构建新增 DMG 输出，解决只读卷场景下自动更新失败的问题。
- **设置页优化** — 新增多厂商选择器、提示词模板编辑器；优化左右修饰键（L/R Ctrl、Shift 等）的 keycap 显示为柔和上标样式。

## v1.1.0 (2026-05)

- **设置主页** — 新增主页，包含使用统计、活跃热力图和输入历史，快速了解语音输入使用情况。
- **统一打包命令** — 单条 `pnpm run pack` 命令同时构建 macOS arm64 和 x64，简化发布流程。
- **设置页重构** — 重新设计设置页面，采用侧边栏导航和自动保存，编辑体验更流畅。
- **自动更新修复** — 修复保存配置时自动更新检查被重复触发的问题，避免多余的网络请求。
- **CI 与发布流程** — 统一打包脚本，添加 CI 流水线，重构发布技能，构建流程更可靠。

## v1.0.8 (2026-04)

- **更新安装修复** — 修复 macOS 上自动更新重启后未退出的问题（Electron 41），在 `quitAndInstall()` 后显式调用 `app.quit()`。
- **托盘清理** — 重启安装前销毁系统托盘，避免更新过程中应用卡住。
- **更新 UI 简化** — 将检查更新、下载、安装合并为单个状态驱动按钮，支持进度显示和错误自动恢复。
- **更新诊断** — 新增 Squirrel.Mac 原生更新事件监听和安装流程日志，便于排查问题。

## v1.0.7 (2026-04)

- **启动速度优化** — WebSocket 连接与音频设备初始化改为并行执行，显著缩短从按下热键到开始录音的等待时间。
- **中文识别修正** — 修复 ASR 识别结果中连续中文字符之间出现多余空格的问题。

## v1.0.6 (2026-04)

- **长按说话模式** — 新增 `app.hotkey_mode`，支持 `toggle` 和 `hold` 两种触发方式，并在设置页提供对应配置入口。
- **快捷键精确匹配** — 录制的左右修饰键现在会严格区分，左/右 `Ctrl`、`Shift`、`Alt`、`Command` 不再互相触发。
- **状态显示修正** — 浮窗会在音频真正开始发送后才进入绿色可用状态，显示时机更贴近真实录音状态。
- **长按短按稳定性** — `hold` 模式短按取消时，不再因为 ASR 连接尚未建立完成而抛出误导性的 WebSocket 异常。
- **启动速度优化** — 调小音频分片大小，缩短开始录音时的体感等待时间。
- **设置页整理** — 原“常规”拆分为“快捷键”和“App 配置”，简化热键提示，并调整配置文件路径的展示方式。

## v1.0.5 (2026-04)

- **Windows 修复** — 修复 Windows 安装后启动报错 "not a valid Win32 application" 的问题。原因是 macOS 编译的 `uiohook-napi` 原生模块被打包进 Windows 安装包，新增 `prepack:win` 脚本在打包前自动清理 macOS 编译产物，确保运行时加载正确的 Windows 预编译模块。

## v1.0.4 (2026-04)

- **主题切换** — 设置页新增浅色 / 深色 / 跟随系统主题偏好，通过 `app.theme` 持久化
- **统一配色** — 新建 `theme.css` 作为唯一配色来源，overlay 与设置窗口共用
- **设置页优化** — 开关布局重构（标题居左、开关居右、字段路径居下），主题选择改为按钮组，精简冗余描述
- **浮窗修复** — 隐藏时重置卡片尺寸，避免显示旧尺寸闪烁
- **构建修复** — 禁用 Windows 原生模块重建，修复启动崩溃和 ASR 配置错误处理
- **代码质量** — 新增 @biomejs/biome 代码检查和格式化工具

## v1.0.3 (2026-04)

- **自定义快捷键录制** — 新增基于 `uIOhook` 的快捷键录制，支持组合键配置
- **开机自动启动** — 设置页新增登录后自动启动开关
- **剪贴板保留** — 新增 `app.keep_clipboard`，粘贴后可保留识别结果在剪贴板
- **句尾句号清理** — 新增 `app.remove_trailing_period`，自动去除最终结果末尾的 `。` / `.`
- **配置模板更新** — `config.yaml.example` 已补充默认热键和新的 app 级配置项
- **文档修正** — README 描述已与当前 macOS / Windows 行为保持一致
